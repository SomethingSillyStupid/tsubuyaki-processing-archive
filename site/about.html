About

#つぶやきProcessing Archive

This is an attribution-forward archive of public p5.js sketches posted on X with the #つぶやきProcessing hashtag.

Each entry keeps a local runnable copy of the sketch code, a small preview for browsing, and direct links back to the original post and artist profile. The archive does not claim ownership of the work; it is a preservation and discovery layer for a public creative-coding practice.

Removal requests

If you are an artist represented here and want an entry removed, please open an issue in the GitHub repository with the post URL or sketch ID. Removal requests should be honored promptly.

Attribution

Cards and detail pages prioritize the X username, original post link, and artist profile link. Sketches are not renamed by the archive.

Feeds

The archive publishes RSS and latest JSON feeds for recent additions.

Built as a static public archive on GitHub Pages.
